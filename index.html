Reading colors

Any CSS Color 4 color should work*:


		let slategray = new Color("slategray");

		// You can use properties to modify coordinates
		// of any other color space (in this case LCH chroma) and convert back
		slategray.chroma *= 1.2; // saturate 20%

		// Output in P3
		slategray.to("p3").toString({precision: 3}); // color(display-p3 0.442 0.501 0.572)

Pass a color space id and coordinates:


		let limer = new Color("p3", [0, 1, 0]);
		limer.spaceId = "srgb"; // Convert to sRGB
		limer + ""; // rgb(0% 98.388% 16.188%)
