Reading colors

Any CSS Color 4 color should work:


			// Any CSS color should work
			let color = new Color("slategray");
			let color = new Color("color(display-p3 0 1 0)");
			let color = new Color("lch(50% 80 30)");

Manipulating colors

You can use properties to modify coordinates of any other color space (in this case LCH chroma) and convert back 


			color.chroma *= 1.2; // saturate 20%

Converting between color spaces & stringifying

Output in any color space


			color + ""; // default stringification
			color.to("p3").toString({precision: 3}); // color(display-p3 0.442 0.501 0.572)

Clip to gamut or don't


			let color = new Color("p3", [0, 1, 0]);
			color.to("srgb").toString({inGamut: false})

Change color space:


			color.space = "srgb"; // Convert to sRGB
			color.spaceId = "srgb"; // Same

Interpolation

Interpolate and get steps by max deltaE number of steps, or both:


			let color = new Color("p3", [0, 1, 0]);

			// Discrete steps
			color.steps("red", {space: "lch", outputSpace: "srgb", delta: 3, steps: 10});

			// Function that accepts a percentage
			let redgreen = color.range("red", {space: "lch", outputSpace: "srgb", delta: 3, steps: 10});
			redgreen(.5); // midpoint

			// Other way to get midpoint
			let redgreen = color.mix("red", .5, {space: "lch", outputSpace: "srgb", delta: 3, steps: 10});

			// Static syntax (every instance method has a static one too)
			Color.mix("color(display-p3 0 1 0)", "red", .5);
