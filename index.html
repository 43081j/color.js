Color.js

A swiss army knife for all your color needs

  • Fully color space aware, not just using color spaces as output
  • Fully up to date with CSS Color 4
  • Readable, object-oriented API
  • Fully extensible
  • From Lea Verou (co-editor of CSS Color 5) and Chris Lilley (co-editor of CSS Color 3, 4, and 5 and ICC member)

Reading colors

Any color your browser supports should work:


		let slategray = new Color("slategray");
		slategray.chroma *= 1.2; // saturate 20%
	

	let blue = new Color("P3", [0, .5, 1]);
	let blue = Color.parse("blue");
	blue.lab // [l, a, b]
	blue.lightness *= 1.2; // make 20% lighter
	blue.a98rgb
